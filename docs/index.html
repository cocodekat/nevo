Welcome to the nevo documentation, this section will guide you trough installing nevo.

MacOS

First, locate to the downloads page. You can find this at the nevo-lang repo by cocodekat or visiting cocodekat.github.io/nevo. From there, download the latest version of nevo.
After installing, unzip the downloaded file.
Then in there, run install.sh. This will download nevo to your computer.

To check if the installation went correctly, type this in the terminal: 
$HOME/nevo/nevo

This should print something about how to use it. If it does, congrats! You installed nevo correctly!

After installing, unzip the downloaded file.
Then in there, run install.bat. This will download nevo to your computer.
