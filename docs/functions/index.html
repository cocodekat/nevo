Functions

Released on 2025-11-27

Why do we have functions?

Functions are a very important part in code. It allows us to easily reuse code instead of having to rewrite it over and over again.

How do we use it?

Now, let’s try it out ourselfs. FIrst, we need to create a function. To do this we also need a type. This can be any type like you saw in the printing doc. The type means what the function will return. If the function is an Integer we at the end of the function do return 1. This tells the code the function exited without problems. Here’s an example: 

int say_hello() { }

Everything inside of the brackets {} is part of the function, to declare parameters we do it exactly the same as in C.

For example, if we have a function called yap() and we want it to have a string parameter that it will print, we can do: 
int yap(char* text) { print("%s\n", text)! return 1! }
