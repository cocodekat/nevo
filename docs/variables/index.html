Variables

Released on 2025-11-18

Why do we have variables?

If you want to store a value, variables are key. They store values in your memory which allows u to get that value anytime. We of course also have variables in nevo.

How do we use it?

Now let’s try it out ourselves. Variables work almost the same as in C, tho there are waaaay more boolean types and typing is NOT needed. It’s optional.

Let’s say we want to define x = 5, we can do it like this: 
x = 5!

This will assign 5 to the variable called x, variables cannot start with numbers or have the same name as anything that is already defined. If we now do 
x = 6!

This will still work. The compiler knows x is already defined so instead of redefining it changes the value.
