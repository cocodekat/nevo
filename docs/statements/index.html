Statements

Released on 2025-12-4

Why do we have statements?

Statements allow us to execute code only if a statement is true/false. This can be super usefull in developing any kind of tools

How do we use it?

Now, let’s try it out ourselfs. First, we need to write the statement. In Nevo we have 2 kind of statements: 
if()
and 
unless()

The if statement runs the code only if the statement is true. The unless statement runs the code UNLESS the statement is true.
Let's try to make a simple if statement. We will make X = 5 and Y = 6, then check if Y is greater then X. 

x = 5!
y = 6!
int main() {
    if(y > x) {
        print("Y is greater then X")!
    }
}

Now try running the code. It should print that Y is greater then X.

Then let's try out the unless() statement. We will do the exact same but then print something unless Y is not greater then X (This is basically the same) 

x = 5!
y = 6!
int main() {
    unless(x <= y) {
        print("Y is greater then X")!
    }
}
